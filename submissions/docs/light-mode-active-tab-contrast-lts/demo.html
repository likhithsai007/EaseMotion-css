Active Tab Visibility Showcase

This demo showcases the fix for the active tab visibility bug in the top navigation bar when switching themes. Use the theme toggle button (🌓) in the top-right corner to switch between dark and light modes.

In the bug, the active navigation tab's text and background colors are hardcoded, causing them to blend into the light mode page background. In this fix, we replace the hardcoded colors with theme variables so that the active tab's contrast stays perfect in both modes.
