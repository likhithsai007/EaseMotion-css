# Scroll Counter Component

A scroll-triggered animated counter component for statistics sections, dashboards, landing pages, and analytics interfaces.

## Features

- Count-up animation from zero
- Scroll-triggered activation
- Intersection Observer implementation
- Prefix support
- Suffix support
- Multiple speed variants
- Fade + slide entrance animation
- Reduced motion support
- Responsive layout
